# auth0-POC

## Deps
* Go.

## Running the service
* Run `go get ./...` to install all dependencies.
* Run the command `go run main.go`.

## Features
* Universal Login.
* MFA(Provided by Auth0 From Dashboard).
* SPA + API Integration.
* JWT Middleware and basic verification of token.
